# Bash Scripting for Linux Automation

This project contains basic Bash scripts created to understand Linux systems and automate common administrative and security-related tasks.

## Purpose
- Learn Bash scripting fundamentals
- Understand Linux system behavior
- Build automation skills useful in Cyber Security

## Scripts Included
- system_info.sh → Displays system details
- user_check.sh → Shows logged-in users
- disk_monitor.sh → Monitors disk usage
- process_monitor.sh → Lists top running processes

This project is part of my long-term Cyber Security learning journey starting from school life.
